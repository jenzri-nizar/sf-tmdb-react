# Symfony Tndb React

Test Application

## Installation

Install env docker
```bash
make start 
```
Run composer install 
```bash
make install 
```

Run DataBase Migration
```bash
make migrations 
```

Run fixtures data
```bash
make fixtures 
```

Install package npm 
```bash
cd htdocs
yarn install
```

Run app react
```bash
npm run watch 
```
